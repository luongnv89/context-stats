Brand Identity

ContextStats

Understand how you use Claude Code — and spend less doing it.

Design Concept
The context window, measured
The frame — the context window: your total token budget
The fill — tokens consumed, green fading to cyan as it grows
The cursor — your current position in the window
The limit — remaining capacity before the edge

The mark is a context window drawn literally — a rounded frame holding a fill gauge. It is the one thing the tool exists to show you: how much of your window is spent, and how much remains.

The fill sweeps from green to cyan as consumption grows, a cursor marks where you are right now, and an indigo tick at the far edge holds the line you are spending toward. Read left to right, it is a live progress bar.

The color progression moves from green (healthy, plenty left) through cyan (filling, mid-session) to indigo (the limit, the edge). The same palette the statusline uses to warn you in real time.

Typography sets Inter Bold as a single word — Context in light slate, Stats in green — so the name reads as one mark while the accent ties back to the gauge.
